Module Common.Location

This module defines the located type which represents data wrapped with a source location. It also has utilities for manipulating located records.

Note that some of the functions here are automatically derived using the Visitors package among others. As such these are not documented. These include functions like show_located, equal_located and fold which function like their Haskell counterparts.

type source_loc = Stdlib.Lexing.position * Stdlib.Lexing.position

This datatype is used to wrap our ASTs with source location information.
